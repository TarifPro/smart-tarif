# smart-tarif

Умный подбор мобильных тарифов: сравнение реальной стоимости, учёт скрытых платежей и ссылки на подключение.

**Бот:** http://t.me/Tarif_Pro_Bot  
**Техподдержка:** 1.tarifpro@gmail.com

Сайт публикуется через GitHub Pages (ветка gh-pages).

## Быстрый старт

1. Клонируйте репозиторий.
2. Внесите изменения в `index.html` и `style.css`.
3. Пушьте в ветку `gh-pages` для публикации.

## Файлы
- `index.html` — главная страница.
- `style.css` — стили.
- `script.js` — мелкие скрипты.
- `privacy.html` — политика конфиденциальности.
- `terms.html` — условия использования.
